FITUR SKELETON LOADER
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✅ Content-Aware Structure
   - 2 area groups (seperti konten asli)
   - 3 device cards per area
   - Mirror struktur card sebenarnya

✅ Device Card Skeleton Components:
   - Header: Icon box + Name placeholder + Status badge
   - Metrics: 3 metric boxes (Lembap, Suhu, Baterai)
   - Footer: Update time + Detail button

✅ Neumorphism Styling:
   - Inset shadows untuk pressed elements
   - Embossed shadows untuk card containers
   - Consistent dengan design system

✅ Animation:
   - Tailwind animate-pulse (pure CSS)
   - Smooth dan tidak mengganggu
   - No JavaScript overhead

✅ Responsive:
   - 1 column (mobile)
   - 2 columns (tablet)
   - 3 columns (desktop)

   Menampilkan multiple items (6 cards total)
