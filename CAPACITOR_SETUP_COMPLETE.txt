🎯 NEXT STEPS - BUILD FIRST APK
═══════════════════════════════════════════════════════════════════

STEP 1: Update Backend URL (PENTING!)
  
  Edit: capacitor.config.json
  
  Ganti IP "192.168.1.100" dengan IP komputer Anda:
  
  Cari IP Anda:
    Linux: ip addr show | grep "inet "
    Windows: ipconfig
    Mac: ifconfig | grep "inet "
  
  Contoh hasil: 192.168.1.50
  
  Update config:
    "url": "http://192.168.1.50:8000"
  
  Lalu sync:
    npx cap sync

STEP 2: Setup Firebase Cloud Messaging (FCM)
  
  A. Buat Firebase Project:
     1. Buka: https://console.firebase.google.com
     2. Create New Project: "AgriNex Smart Drip"
     3. Add Android App:
        - Package name: com.agrinex.smartdrip
        - Download google-services.json
     
  B. Install google-services.json:
     cp ~/Downloads/google-services.json android/app/
     
  C. Get FCM Server Key:
     1. Firebase Console → Project Settings → Cloud Messaging
     2. Copy "Server key"
     3. Add to Laravel .env:
        FCM_SERVER_KEY=your_server_key_here

STEP 3: Start Laravel Server
  
  php artisan serve --host=0.0.0.0 --port=8000
  
  (Jangan gunakan localhost, gunakan 0.0.0.0 agar accessible dari device)

STEP 4: Build APK
  
  A. Open Android Studio:
     npx cap open android
     
  B. Wait for Gradle sync (first time ~3-5 menit)
  
  C. Build APK:
     Menu: Build → Build Bundle(s) / APK(s) → Build APK(s)
     
  D. Lokasi APK:
     android/app/build/outputs/apk/debug/app-debug.apk

STEP 5: Install & Test di Device
  
  A. Enable USB Debugging di Android:
     Settings → About Phone → Tap "Build number" 7x
     Settings → Developer Options → USB Debugging ON
     
  B. Connect via USB dan install:
     adb install android/app/build/outputs/apk/debug/app-debug.apk
     
     Atau copy APK ke device dan install manual

STEP 6: Test Push Notifications
  
  A. Login di app
  
  B. Test via browser (buka Postman/Insomnia):
     POST http://192.168.1.50:8000/api/v1/mobile/test-notification
     Headers:
       Authorization: Bearer YOUR_AUTH_TOKEN
       Accept: application/json
     
  C. Notification should appear!

           'Lahan A membutuhkan irigasi'

   }

🔥 COMMON ISSUES & SOLUTIONS
═══════════════════════════════════════════════════════════════════

Issue 1: App terbuka tapi blank/white screen
Solution:
  - Cek IP di capacitor.config.json sudah benar
  - Cek Laravel server running di 0.0.0.0:8000
  - Cek device dan laptop di network yang sama
  - Buka Chrome DevTools remote: chrome://inspect

Issue 2: Push notification tidak muncul
Solution:
  - Pastikan google-services.json sudah di android/app/
  - Pastikan FCM_SERVER_KEY di .env sudah benar
  - Cek Logcat di Android Studio untuk error FCM
  - Test dulu di foreground, baru background

Issue 3: Cannot connect to backend
Solution:
  - Jangan pakai "localhost", pakai IP
  - Cek firewall laptop tidak block port 8000
  - Test buka browser di HP: http://192.168.1.X:8000

Issue 4: Gradle build failed
Solution:
  - Pastikan Java 17 atau 21 installed
  - Update Android Studio ke versi terbaru
  - Delete android/.gradle dan rebuild

Issue 5: Service Worker tidak register (di browser)
Solution:
  - Buka HTTPS atau localhost saja (SW butuh secure context)
  - Cek browser console untuk error
  - Unregister SW lama: chrome://serviceworker-internals
