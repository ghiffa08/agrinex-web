🔧 KEY OPTIMIZATIONS
══════════════════════════════════════════════════════════════════════════════

✅ SCHEMA MAPPING FIXED
   recorded_at (bukan received_at)
   temperature (bukan temperature_c)
   soil_moisture (bukan soil_moisture_pct)
   started_at, stopped_at (bukan waktu_mulai, waktu_selesai)
   water_used_liters (bukan total_air_liter)

✅ N+1 PREVENTION
   Eager loading: ->with('device:id,name,lokasi')
   Count queries: ->withCount(['sensorData', 'irrigationLogs'])

✅ EXPORT STYLING
   Professional Excel headers dengan color-coding
   Auto-size columns untuk readability
   Title + date metadata di sheet

✅ USER EXPERIENCE
   Quick date presets (7/30/90/180 hari, bulan ini)
   Preview sebelum download (tampilkan ringkasan)
   Loading modal dengan spinner
   Neumorphic shadows konsisten

✅ ERROR HANDLING
   Validation di controller
   Try-catch di service
   Logging dengan trace
